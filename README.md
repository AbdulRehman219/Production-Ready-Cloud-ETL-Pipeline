# 🚀 Cloud ETL Pipeline Project

## 📌 Overview
This project demonstrates a production-style ETL pipeline that extracts Bitcoin price data from a public API, transforms it, and loads it into AWS S3.

## 🧱 Architecture
API → Python → Local Storage → AWS S3

## 🛠 Tech Stack
- Python (Pandas, Requests)
- AWS S3 (Cloud Storage)
- ETL Pipeline Design

## ⚙️ Features
- Modular pipeline (Extract, Transform, Load)
- Error handling
- Cloud integration (S3)
- Local backup storage
